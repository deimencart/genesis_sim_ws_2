# CLAUDE.md — Genesis Denim Experiments

Este repo es CÓDIGO DE TRABAJO, no el wiki de conocimiento. El wiki vive en
~/Genesis_Simulator_KB (repo separado) — ahí están los papers, la justificación
de parámetros, y el ROADMAP.md con las 6 fases.

Antes de escribir código para una fase:
1. Lee el README.md de la carpeta de esa fase (si existe) para ver qué ya se probó.
2. Si necesitas contexto de parámetros/decisiones, consulta las notas relevantes
   del vault (ruta: ~/Genesis_Simulator_KB) — no las reescribas aquí, solo cítalas.
3. Cada script exitoso o hallazgo importante se documenta en el README.md de su
   carpeta de fase, en 2-3 líneas.

Al terminar una fase o un experimento importante, avísame para actualizar el
ROADMAP.md del vault (marcar checkbox) y decidir si el snippet final debe
copiarse a 06_Code_Snippets/ del vault como versión validada y reutilizable.
